# Compliance Overview

Borderless Payroll Copilot implements compliance-aware architecture hooks for enterprise workflows while keeping on-chain payroll settlement auditable.

## Included
- KYC/KYB workflow integration points
- AML/sanctions alert hooks
- Large payout threshold detection
- Manual approval threshold signaling
- Governance policy persistence

## Not Claimed
- Native sanctions provider certification
- Native KYB provider adjudication

These must be integrated through external compliance vendors.
