# Operational Governance

## Governance Domains
- Treasury ownership and dual-control
- Approval thresholds
- Compliance alert triage
- Incident command roles
- API key lifecycle governance

## Required Controls
- OWNER/ADMIN separation for production key rotation
- Periodic review of compliance alerts
- Mandatory reconciliation review before payout window close
- Monthly disaster recovery drill

## Audit Requirements
- Log all credential lifecycle events
- Log partner webhook subscription changes
- Preserve reconciliation artifacts for due diligence
